# Data_Entrying-FORTEX-6.0-Final
Analisis klaster danau menggunakan algoritma HDBSCAN untuk mengelompokkan danau berdasarkan kualitas air dan karakteristik DAS. Tujuannya adalah mengidentifikasi pola dan memprioritaskan konservasi. Evaluasi klaster menggunakan metrik Silhouette Coefficient dan DBI. Temuan: Kualitas air menurun akibat tekanan pertanian/urbanisasi.
